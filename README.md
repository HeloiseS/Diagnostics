# Diagnostics

This repository contains code and data used in Dorn-Wallenstein & Levesque (2018). For examples of how to use the code contained in [code/]. The main piece of code is [`diagnostics.py`](./code/diagnostics.py). [paper/diagnostics.ipynb](./paper/diagnostics.ipynb) is an example notebook that imports functions from [`diagnostics.py`](./code/diagnostics.py), and shows some examples of how to use them.
